Introduction

JavaScript is a powerful scripting language for web development.

It runs in the browser and allows you to create dynamic websites.

console.log("Hello World");
  • Client-side
  • Dynamic
  • Flexible
Variables

Variables are used to store data values in JavaScript.

You can declare variables using var, let, or const.

let name = "Alice";
  • var: function scoped
  • let: block scoped
  • const: block scoped and immutable
Functions

Functions are blocks of code designed to perform tasks.

They can accept parameters and return values.

function greet(name) { return "Hello " + name; }
  • Reusable
  • Parameterized
  • Return values
Loops

Loops are used to repeat code multiple times.

JavaScript provides several types of loops.

for (let i = 0; i < 5; i++) { console.log(i); }
  • for loop
  • while loop
  • do...while loop
Arrays

Arrays are used to store multiple values in a single variable.

They are zero-indexed and can contain mixed types.

const fruits = ["apple", "banana", "cherry"];
  • Access by index
  • Length property
  • Array methods
